# Store Listing: UI Friction Detector

## Description
Stop guessing and start detecting. UI Friction Detector is a powerful real-time UX audit tool that lives in your browser. It scans any webpage for friction points that kill conversion and frustrate users.

Built for UX designers, product managers, and developers who want immediate, actionable feedback without running heavy audits.

## Key Audit Modules
- 🎯 **CTA Clarity**: Flags vague or overly wordy buttons.
- 🎨 **Color Contrast**: WCAG 2.1 compliance check.
- 📐 **Tap Targets**: Ensures mobile usability with 44px minimums.
- ⚡ **Interaction Delay**: Monitors FID and TTFB.
- 🧱 **Cognitive Load**: Detects density issues that overwhelm users.
- 📏 **Layout Stability**: Flags CLS issues before they bother users.
- 🧠 **Psychology Patterns**: Identifies infinite scroll traps and overlay fatigue.

## Features
- **Instant Highlighting**: See exactly where the friction is.
- **AI UX Advisor**: Get expert fixes powered by Claude.
- **Roast Mode**: Turn on the heat with savage UX feedback.
- **Production Ready**: Zero dependencies, lightweight, and fast.

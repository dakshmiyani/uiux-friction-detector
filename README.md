# UI Friction Detector: AI-First UX Auditor ⚡

**Stop guessing. Let AI find every friction point on your website in seconds.**

UI Friction Detector is a high-performance Chrome Extension that transforms your browser into a world-class UX Consultant. Powered by **Groq (Llama-3.3-70b-versatile)**, it performs deep, context-aware audits to discover conversion killers, accessibility gaps, and usability friction that traditional tools miss.

---

## ✨ Key Features

### 🧠 Deep AI Scan (AI-First)
Unlike rule-based scanners, this extension extracts a full **metadata snapshot** of your page—including CTAs, images, forms, and layout structure—and sends it to a specialized UX Expert model to **discover** frictions dynamically.

### 😈 Brutally Honest AI Auditor
Equipped with a "Brutally Honest" persona, the AI provides ruthless, actionable feedback from the perspective of a senior UX Designer & CRO Expert. Toggle **Roast Mode** for savage, witty critiques that make UX improvements memorable.

### 🎯 Neon Glow Highlighting
Pinpoint exactly where changes are needed. AI-detected issues are mapped back to the DOM with a high-visibility neon glow animation and detailed tooltips, regardless of scroll position or page complexity.

### 📄 Professional HTML Reports
Generate a client-ready **HTML Audit Report** with one click. These styled reports include overall UX scores, categorical breakdowns, and clearly formatted findings with specific "Savage Roasts" and "AI Fix Suggestions."

### 🔄 Real-Time Tab Sync
Automatically detects when you switch tabs and warns you if results are stale. The proactive "Stale Results" overlay ensure you're always auditing the right page.

## 🚀 Getting Started

### 1. Installation
1. Clone this repository or download the ZIP.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** (top right).
4. Click **Load unpacked** and select the `ui-friction-detector` folder.

### 2. Connect Groq (Recommended)
This extension is designed to be **AI-First**. To unlock the full power:
1. Get a free API key at [console.groq.com](https://console.groq.com).
2. Open the extension and click the **⚙️ Settings** icon.
3. Paste your `gsk_` key and click **Save**.
